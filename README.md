# Cataso

カタソ統合リポジトリ。

This repository integrates the frontend and backend of Cataso.

## Origins

- `frontend/` is derived from `mitdok/tkmonline`
- `backend/` is derived from `mitdok/tkmninja`

The original repositories are preserved for attribution and upstream reference.

## Structure

- `frontend/`: static client / GitHub Pages frontend
- `backend/`: Node.js WebSocket server
- `docs/`: protocol, deployment notes, upstream notes
